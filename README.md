# Ignite Growth Landing Page

A simple HTML landing page for Ignite Growth that directs users to the platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your logo to the `public/` directory as `logo.png`

3. Run the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Pages

- `index.html` - Main landing page
- `pricing.html` - Pricing information
- `learn-more.html` - Detailed information about services
- `privacy-policy.html` - Privacy policy
- `terms-of-service.html` - Terms of service

## Deployment

The site is configured for Vercel deployment. The build output will be in the `dist/` directory.

## Notes

- Make sure to add your logo image to `public/logo.png`
- Update contact email addresses in the HTML files if needed
- The platform link currently points to `https://bd.ignitestrategies.co` - update this when the nextgen app is ready
